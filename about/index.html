heard Ubuntu, tried and feel sucks, tried CentOS, and complete my Linux 101

then there is Debian, love it, my favorate until I met Arch few years later,

how to do it at offical Deiban CustomCD tutorial

gradually I learned to used some hacking tools, became a script kiddie

those that I remember: Metasploit, Exploit-DB, Nmap, some about mysql injection,
and some about wireless cracking(Aircrack-ng, etc)

then I know the term “penetration test”, and wanna be a Certifcated Ethical(White Hat) Hacker

I realised using tools and familiar with linux is not enough, so I start to read Intel CPU Manual,
tons of Assembly language books and Computer Architecture books, then the ELF Standard, learned
how to disassembly manually, convert C to assembly back and forth, finally write a ELF Executable
file with hexadecimal with bvi.
